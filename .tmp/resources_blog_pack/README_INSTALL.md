# Resources + Blog Pack (CerebraTechAI)

This bundle contains:

1. `blog.ts` (updated) — removes exaggerated claims and adds 6 practical posts.
2. `resources.ts` (updated) — upgrades TXT placeholders to real downloadable assets + adds 5 new resources.
3. `public/resources/*` — the actual asset files referenced by `resources.ts` (PDF/ZIP/XLSX).

## How to install in your repo

1. Copy `public/resources/` into your project at:
   - `<your-repo>/public/resources/`

2. Replace (or merge) your data files:
   - `<your-repo>/path/to/blog.ts` <-- replace with `blog.ts` from this bundle
   - `<your-repo>/path/to/resources.ts` <-- replace with `resources.ts` from this bundle

If your project stores these files elsewhere, search for:

- `export const BLOG_POSTS`
- `export const RESOURCES`

## Notes (no hype / not over-claiming)

- Titles and descriptions avoid numeric performance claims and “X+ deployments”.
- PDFs include a disclaimer: templates / best-practice guidance, not legal advice or certification claims.
