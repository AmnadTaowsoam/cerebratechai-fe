Edge AI Deployment (Outline)
----------------------------
TH:
- เลือกฮาร์ดแวร์ตามข้อจำกัด: latency, พลังงาน, อุณหภูมิ, connectivity
- วางแผน data flow: buffering, offline mode, retry, sync
- ทำ model optimization: quantization/ONNX/TensorRT (ถ้าเหมาะสม)
- Monitoring ที่หน้างาน: log, metrics, watchdog, remote update
- ความปลอดภัย: device identity, key rotation, least privilege
- แผนบำรุงรักษา: OTA update, version pinning, rollback

EN:
- Choose hardware by constraints: latency, power, heat, connectivity
- Plan data flow: buffering, offline mode, retry, sync
- Model optimization: quantization/ONNX/TensorRT when appropriate
- On-device monitoring: logs, metrics, watchdog, remote updates
- Security: device identity, key rotation, least privilege
- Maintenance: OTA updates, version pinning, rollback
