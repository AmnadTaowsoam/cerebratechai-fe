MLOps Templates (Text Pack)
--------------------------
A) Monitoring Checklist
- System health: uptime, error rate, latency
- Data pipeline: freshness, schema changes, missing values
- Model/LLM quality: sampled reviews, regression tests
- Drift signals: input distribution shifts, outcome changes
- Alerts: thresholds + ownership + escalation path

B) Incident Runbook (Skeleton)
1) Detect: What alert / symptom?
2) Triage: impact, affected users, severity
3) Mitigate: rollback / disable feature / fallback mode
4) Communicate: status page / internal updates
5) Resolve: root cause + fix + verify
6) Prevent: tests, monitoring, docs update
